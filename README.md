## Features

* AngularJS 1.5.x with ES2015 flavour
* Twitter Bootstrap with Sass
* Gulp
* Webpack
* Long term caching
* Karma, mocha and Sinon.js 
* Istanbul test coverage
* ESLint, HTMLHint
* TravisCI, Heroku, Continuous deployment
